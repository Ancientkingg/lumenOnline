You're able to download the default atlas for 20w11a here!

SpiderEye

Alternatively you can use SpiderEye, a program by SirBenet to retrieve the atlas yourself. If it does not work try:

Launch SpiderEye. Select the minecraft launcher or a link to it. Then in the Minecraft launcher go to
Installations -> Edit (on ··· menu of profile you want to use) -> More Options
and setting Java Executable to:
C:\Program Files (x86)\Minecraft Launcher\runtime\jre-x64\bin\javaw.exe
